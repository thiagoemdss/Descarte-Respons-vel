# Descarte-Respons-vel
Aplicação web desenvolvida para conscientização ambiental sobre o descarte correto de resíduos urbanos. O projeto apresenta:  busca dinâmica  informações oficiais sobre riscos ambientais  dicas de descarte adequado  links úteis para Ecopontos  interface moderna com efeito vidro fosco  Criado como parte do meu portfólio em desenvolvimento web.
